OPTION 3: Tester un paiement complet
-------------------------------------
1. Ouvrir: http://localhost:3000/checkout.html?orderId=TEST-002&amount=10000

2. Sélectionner opérateur (ex: MTN)

3. Entrer numéro: 97 00 00 00

4. Cliquer "Payer 10 000 XOF"

5. Page "Confirmez sur votre téléphone" s'affiche

6. Mode simulation:
   - Reste en "pending"
   - Utilisez les boutons de test (si vous les ajoutez)

7. Mode réel (si FedaPay activé):
   - Vous recevez notification USSD
   - Entrez code PIN
   - Paiement validé ou refusé

================================================================================
                        API DISPONIBLES
================================================================================

1. GET /health
   → Status serveur

2. GET /api/plans
   → Liste des 3 plans BeninPay

3. POST /api/checkout
   → Créer URL de paiement
   Body: { orderId, amount, shopName, customerName, items }

4. GET /api/checkout/test
   → URL de test avec données fictives

5. POST /api/payment/initiate
   → Initier paiement FedaPay
   Body: { amount, phone, operator, orderId }

6. GET /api/payment/status/:id
   → Vérifier statut transaction

7. POST /api/payment/webhook
   → Webhook FedaPay (notifications)
