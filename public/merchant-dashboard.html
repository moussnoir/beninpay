🏪 Bouton sur ma Boutique Shopify
Integration

Ajoutez le bouton "Payer avec Mobile Money" sur la page panier de votre boutique. Vos clients pourront payer tous leurs articles d'un coup via Mobile Money!

CODE A COPIER DANS VOTRE THEME SHOPIFY:
Installation en 3 etapes:
1. Allez dans votre admin Shopify > Boutique en ligne > Themes > Actions > Modifier le code
2. Ouvrez le fichier theme.liquid (dans Layout)
3. Collez le code juste avant </body>

Le bouton apparaitra automatiquement sur la page Panier (/cart).
Le client ajoute ses produits au panier, puis clique "Payer avec Mobile Money" pour payer le total avec MTN, Moov ou Celtis.
