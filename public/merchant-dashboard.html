Intégration

Ajoutez le bouton "Payer avec Mobile Money" sur la page panier de votre boutique Shopify. Vos clients pourront payer tous les articles de leur panier via MTN, Moov ou Celtis!

CODE À COPIER DANS VOTRE THÈME:
Installation en 3 étapes:
1️⃣ Admin Shopify → Boutique en ligne → Thèmes → Actions → Modifier le code
2️⃣ Ouvrir le fichier theme.liquid (dans Layout)
3️⃣ Coller le code juste avant </body>

✅ Le bouton apparaît automatiquement sur la page Panier (/cart).
Le client ajoute ses produits puis clique "Payer avec Mobile Money" pour payer le total.
